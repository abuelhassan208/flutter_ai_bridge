OpenAIProvider

OpenAIProvider class

OpenAI provider implementation.

Supports GPT-4o, GPT-4, GPT-3.5-turbo, and other OpenAI models.

final openai = OpenAIProvider(
  config: AIConfig(
    apiKey: 'sk-...',
    model: 'gpt-4o',
  ),
);

final response = await openai.complete([
  AIMessage.user('Hello!'),
]);
