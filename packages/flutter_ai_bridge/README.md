# flutter_ai_bridge

The umbrella package for the Flutter AI Bridge ecosystem — re-exports `ai_bridge_core` and `ai_bridge_ui` for a single-import developer experience.

## Quick Start

```dart
import 'package:flutter_ai_bridge/flutter_ai_bridge.dart';

// All core models, agents, pipelines, RAG, and UI widgets
// are available from this single import.
```

See the full documentation at [ai_bridge_core](https://pub.dev/packages/ai_bridge_core).
